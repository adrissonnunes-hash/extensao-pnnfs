# Livros ISS.Net Santa Maria

Projeto em Python para automatizar, com um único login manual do contador, a geração e o salvamento dos documentos oficiais do ISS.Net de Santa Maria para uma lista de CNPJs.

Versão enviada ao Codex: **32.0.0**.

## Objetivo imediato

Otimizar o processamento em lote para muitos CNPJs, especialmente:

- localização rápida dos CNPJs no portal;
- troca rápida entre empresas;
- identificação rápida das pastas dos clientes em `O:\`;
- download dos PDFs oficiais sem depender de foco, nova guia ou clique manual;
- preservação integral das regras de segurança das pastas dos clientes.

## Arquivo do projeto

O código-fonte completo está em `Livros_ISSNet_Santa_Maria_v32.zip`.

No Windows, execute `EXTRAIR_PROJETO.ps1` para extrair o conteúdo na pasta `src`.

As instruções detalhadas para o Codex estão em `AGENTS.md`.
